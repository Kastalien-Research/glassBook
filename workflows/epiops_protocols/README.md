# EpiOps Protocols

This bundle contains markdown specifications for the operational epistemic workflows other than Ulysses, using Ulysses as the structural template.

Included protocols:

- Ariadne Protocol — topology discovery before intervention
- Theseus Protocol — invariant-preserving transformation
- Cassandra Protocol — seductive-uncertainty and pre-mortem analysis
- Hephaestus Protocol — minimal reproducible case construction
- Minos Protocol — source-of-truth adjudication
- Hermes Protocol — commitment-signal extraction
- Janus Protocol — rollback and irreversibility planning

Ulysses is intentionally excluded because it was provided separately as the completed template.
